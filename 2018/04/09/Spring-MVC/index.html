@RequestMapping(“/count”)
public ModelAndView check(HttpSession session){
Integer i = (Integer) session.getAttribute(“count”);
if (i == null)
i = 0;
i++;
session.setAttribute(“count”, i);
ModelAndView mav = new ModelAndView(“count”);
return mav;
}

文件上传与下载

问题

实践过程中遇到的问题：

  1. Spring MVC部署到tomcat，分为两步，第一步新建一个文件夹，用于保存编译输出的class文件，通过File->ProjectStruture里设置Modules和Artifacts中的output path到这个文件夹；第二步是新建一个tomcat server并设置web path，url等

  2. 使用文件上传时报错：提示没有发现FileItemFactory，这里需要引用commons-fileupload-1.3.3.jar，下载之后复制到lib目录，鼠标放到该jar包上右键Add as Libary..但是依然报错，解决方法：File->Project Structure-Artifacts，点击右下角Fix按钮即可

  3. 在controller的方法参数里用到了HttpServletRequest和HttpServletResponse，lib目录只有4个spring的jar包，可以通过引用tomcat来导入相关的类，具体做法：File->ProjectStruture->Modules->Dependies，添加tomcat并勾选

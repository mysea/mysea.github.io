配置next主题

发表于 2017-08-15 | 分类于 hexo | 本文总阅读量 次

hexo博客系列第二篇，讲解next主题的一些配置，很多网上已经有了，因此在后面给出两个相对较好的链接，更多配置请参考给出的链接。

设置博客头像

把你的头像文件放在next主题文件夹下(D:\hexo\themes\next\source\images)，然后在next文件夹下的主题配置文件_config.yml中找到avator，如下图修改即可

添加标签和分类

在hexo文件夹下，命令行输入hexo new page tags，然后编辑source\tags\index.md文件，添加type为tags：

title: tags
date: 2017-08-16 11:03:46
type: "tags"

打开主题配置文件(next文件夹下的_config.yml)，取消对应的注释：

然后在你的文章开头添加tags标签即可：

title: 博客排版那些事儿
date: 2017-08-15 20:02:45
categories: "hexo"
tags: [hexo,...,...]

同样，命令行输入hexo new page actegories，然后编辑source\categories\index.md文件，添加type为categories，在主题配置文件取消注释，在文章开头添加categories标签即可

命令行中输入 hexo new page “about”，然后编辑对应的文件，即可创建关于页面

修改主题配置文件

打开主题配置文件(hexo\themes\next_config.yml)，menu下增加resource: /resource/， menu-icons下增加resource:icon-name，icon从这里fontawesome找一个，复制名字即可

修改语言配置文件

打开 hexo\themes\next\languages 目录下的zh-Hans.yml文件，menu下增加 resource: 干货

这样就自定义目录成功了。
